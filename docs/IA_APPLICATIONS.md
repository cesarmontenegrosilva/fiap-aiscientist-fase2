# Possíveis aplicações em IA

A tabela `gold.features_ml` pode ser usada como entrada para estudos futuros, por exemplo:

- prever a chance de um município atingir a meta;
- classificar municípios por nível de vulnerabilidade;
- identificar grupos com comportamento semelhante;
- avaliar quais fatores territoriais estão associados ao indicador;
- priorizar municípios para políticas públicas.

Como o projeto usa somente uma amostra pequena de alunos, a parte de IA é apresentada como possibilidade arquitetural. Um modelo real exigiria dados completos, variáveis socioeconômicas, separação treino/teste e avaliação de viés.
