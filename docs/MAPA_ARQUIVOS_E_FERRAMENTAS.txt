FERRAMENTAS E SERVIÇOS

Google Cloud Platform
  Nuvem escolhida para hospedar a demonstração temporária.

BigQuery
  Consulta as fontes públicas e armazena as camadas Bronze, Silver, Gold e Monitoring.

Cloud Storage
  Armazena Parquet opcional e arquivos temporários do Dataflow.

Pub/Sub
  Recebe as mensagens simuladas de atualização do indicador.

Dataflow
  Executa o pipeline streaming definido com Apache Beam.

Apache Beam
  Implementa a leitura, validação e separação de eventos válidos e rejeitados.

Dataform
  Compila e executa SQLX, dependências e assertions das camadas Silver e Gold.

Terraform
  Cria e destrói a infraestrutura de forma reproduzível.

Cloud Run Jobs
  Opção gerenciada para executar a ingestão batch em contêiner.

Cloud Scheduler
  Agenda o Cloud Run Job quando a execução recorrente é demonstrada.

Cloud Monitoring
  Recebe o alerta de backlog do Pub/Sub.

Artifact Registry
  Armazena a imagem Docker do job batch.

Python
  Implementa ingestão, configuração, streaming, validação e testes.

Pydantic
  Valida o contrato dos eventos streaming.

Node.js, npm e npx
  Executam o CLI do Dataform.

VS Code
  Ambiente local usado para editar e executar o projeto.

Git e GitHub
  Versionam o código e demonstram branches, commits e Pull Requests.

PowerShell
  Executa comandos no Windows e o script principal de exclusão.

Git Bash
  Executa os arquivos .sh no Windows.

Looker Studio
  Pode consumir as consultas da pasta sql/dashboard para criar visualizações.
