FONTES DE DADOS USADAS NO PROJETO

Este arquivo registra a URL e a URI BigQuery de cada fonte coletada pela aplicação.
A ingestão é feita no BigQuery público da Base dos Dados. O projeto do aluno paga
somente o processamento das consultas e armazena as cópias na camada Bronze.
